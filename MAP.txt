Vamos desenvolver o MAKEMAP um aplicativo desktop profissional para criação, planejamento e organização completa de mundos de MMORPG.

O software não possui objetivo de substituir uma Game Engine.
Sua função é permitir que Game Designers organizem visualmente todos os elementos do jogo antes da implementação.
O sistema deverá permitir visualizar a progressão completa do jogador desde o Level 1 até o End Game.
Toda informação será centralizada em um único projeto.

Público-Alvo: Desenvolvedores Indie, Estúdios AAA, Game Designers 

O software deverá permitir organizar:Mundo, Continentes, Reinos, Regiões, Biomas, Cidades, Vilas, Castelos, Dungeons, NPCs, Mobs, Bosses, World Bosses, Recursos, Economia, Quests, Cadeias de Quest, Eventos, Clima, Lore, Facções, Progressão.
Tudo utilizando uma interface extremamente visual.

O software será inspirado nos melhores aspectos de: Inkarnate, World Anvil, Notion, Miro, Figma, Unreal Editor, Civilization VI e Age of Empires Scenario Editor

Tema: Liquid Glass

Inspirado em: Apple VisionOS
